Editor help
Formatting
  • Select text → click toolbar button
  • Bold: Ctrl+B
  • Italic: Ctrl+I
  • Underline: Ctrl+U
Links & Images
  • Link: select text → click 🔗 → enter URL
  • Remove link: click linked text → click 🔗
  • Image (URL): paste URL in field above editor → Insert URL
  • Image (file): click 📷 in toolbar
  • Delete image: click image → Delete
Email tips
  • Use style="..." inline — email clients strip external CSS
  • Max width 600px works best
  • Avoid JavaScript — blocked by all email clients
  • Test in Gmail + Outlook before sending
Variables
  • {{name}} — contact name
  • {{title}} — job title
  • {{company}} — company name
  • {{website}} — website URL
  • {{domain}} — domain only
  • {{location}} — city/region
  • {{ai_summary}} — AI description
Tabs
  • Editor — WYSIWYG
  • HTML source — raw HTML
  • Preview — rendered with variables
  • CSS — email base styles
